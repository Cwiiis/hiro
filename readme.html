Tips

“Mirrored” display settings are recommended

Direct to Rift support is currently buggy, and setting the displlays to be unique works, but requires dragging the browser window to the HMD display, which becomes awkward.

  • Mac: System Preferences > Displays
  • Windows: Display > Screen Resolution

Set Mac display settings to 60hz

Macs exhibit extreme judder at the DK2’s default 75hz refresh rate. Configuring the display preferences to 60hz mostly addresses the problem. Running at 60hz does result in increased latency and pixel persistence (smearing), but in our experience the tradeoff is well worth it.

  • Mac: System Preferences > Displays

Troubleshooting

First make sure you are using a VR-enabled browser like our experimental Firefox builds. Normal Firefox will not work. Get Firefox with VR support here.

Also make sure that your headset is plugged in, and that it is powered on.

If the above are true, first try restarting the browser and reloading this page. Does the problem persist?

If so, try restarting the Oculus Rift SDK service. See: OculusConfigUtil > Tools > Advanced > Restart Service. Then restart the browser again.
